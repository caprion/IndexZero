# M0 Light Rubric

This rubric rewards the quality of reasoning, not whether the student guessed the hidden ranking formula.

## Scoring

Score each category from 0 to 2.

| Category | 0 | 1 | 2 |
| --- | --- | --- | --- |
| Specificity | Vague claims like "it is popular" or "it matches better" | Some concrete language, but still generic in places | Concrete, rank-aware claims tied to the actual result and query |
| Evidence | No visible evidence from the page | Some evidence, but not enough to support every claim | Claims are backed by observable clues: title match, brand, price, ratings, badges, sponsorship, category fit, and so on |
| Alternatives | Treats one explanation as certain | Mentions another possibility once or twice | Regularly names plausible alternative explanations when more than one signal could explain the ranking |
| Uncertainty | Pretends to know everything | Admits uncertainty, but vaguely | Clearly marks what is unknown and where confidence is high, medium, or low |

## How to use it

- **6-8 points:** strong M0 work
- **4-5 points:** acceptable, but too generic in places
- **0-3 points:** incomplete or not really observing the system

## Reminder

M0 is not a correctness test. A student can be wrong about the mechanism and still do strong work if the reasoning is concrete, evidence-based, and honest about uncertainty.
